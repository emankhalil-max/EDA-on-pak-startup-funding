# 🚀 Pakistan Startup Ecosystem Analysis

This project explores the **Pakistan startup landscape** using data analysis and interactive visualization. It uncovers key trends in startup growth, industry distribution, funding patterns, and operational status.

---

## 📊 Project Overview

The goal of this project is to perform **Exploratory Data Analysis (EDA)** on startup data and transform insights into an **interactive Streamlit dashboard**.

---

## 🔍 Key Insights

* Major startup hubs are **Karachi and Lahore**
* **Fintech and E-commerce** dominate the industry landscape
* Startup growth increased significantly after **2015**
* Most startups are currently **active**
* Funding is unevenly distributed across industries

---

## 🛠️ Tech Stack

* **Python**
* **Pandas & NumPy** (Data Cleaning & Analysis)
* **Matplotlib & Seaborn** (Visualization)
* **Streamlit** (Dashboard)
